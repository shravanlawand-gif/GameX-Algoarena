# Code Arena Battles

Battle AI robots by writing code! Learn Python, SQL, and C++ in an epic coding arena game.

## Project Setup

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server

## Environment Variables

Make sure to set up your `.env` file with the required API keys (e.g., `GEMINI_API_KEY` if using the AI challenge generator).
